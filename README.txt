[Android] 알람시계 어플

개발 환경 : Android Studio
사용 기술 : Kotlin, SQLite

동작 구성
1.메인 화면에서 "+" 아이콘을 누르면 알람을 추가하는 timepicker 화면으로 이동

2.시간 설정 및 날짜입력 후 추가

3.메인 화면에 알람이 추가됨 스위치를 키고 해당 시간과 날짜가 일치하면 알람이 울림

4.mission1 액티비티가 실행되고 미션을 클리어하면 알람 종료(메인화면의 알람 스위치 off)

5.알람 삭제 버튼을 누르면 추가된 알람 중 삭제할 알람을 선택 후 삭제

6.알람 관련 정보는 SQLite를 활용하여 휴대폰 내부 DB에 저장되는 형태
